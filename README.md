# CRUD MongoDb

## Installing packages

Install server packages

```bash
  # in the terminal write
  cd server
  npm install

  # Then run
  npm run dev
```

```bash
  # in the terminal write
  cd client
  npm install

  # Then run
  npm run dev
```

```bash
  # in the terminal write
  # in the root folder
  npm install

  # Then run
  npm run dev
```
